# Gridel's Hermes Skills

Hermes Agent 技能集合，专注实用工具。

## 技能列表

### bilibili-content — B站字幕提取 + 评论精华

三策略字幕获取（AI 字幕 → 投稿者上传 → Whisper ASR 兜底），热门评论提取。支持多 P、多格式输出（学习笔记/摘要/博客/时间线）。

```bash
hermes skills install https://raw.githubusercontent.com/gridel-li/hermes-skills/main/bilibili-content/SKILL.md
```

[完整文档](./bilibili-content/SKILL.md)

## 安装

每个技能独立安装。在 Hermes 对话中输入：

```
hermes skills install <SKILL.md 的原始 URL>
```

或直接用上面各技能下的命令。
